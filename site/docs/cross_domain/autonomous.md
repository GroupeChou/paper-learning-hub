# 自主决策系统

> **核心问题**：时序预测如何指导智能体的自主决策？

---

## 🎯 研究方向

### 方向一：预测驱动的决策闭环

```
┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐
│ 时序预测 │ →   │ 决策引擎  │ →   │ 行动执行  │ →   │ 结果反馈  │
│ (模型)   │     │ (Agent)  │     │ (API调用) │     │ (新数据)  │
└──────────┘     └──────────┘     └──────────┘     └──────────┘
       ↑                                               │
       └───────────────────────────────────────────────┘
                    （形成闭环）
```

### 方向二：时序感知强化学习

- 将时序预测作为RL的状态输入
- Agent在动态环境中做出最优决策
- 应用场景：库存管理、运力调度、定价策略

---

## 🔗 与顺丰业务的关联

这正是**预测×规划联动方案**的核心：

| 场景 | 预测角色 | 规划(Agent)角色 |
|------|---------|----------------|
| 干支线网络规划 | 各线路件量预测 | Agent决定车辆数、载重上限 |
| 中转场作业 | 各班次件量预测 | Agent决定人力排布 |
| 航空枢纽 | 流向预测 | Agent决定舱位分配 |
