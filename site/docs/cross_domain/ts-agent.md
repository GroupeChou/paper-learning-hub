# 时序预测 Agent

> **核心问题**：如何构建一个能够自主进行时序分析与预测的智能体？

---

## 🎯 研究方向

### 方向一：LLM 作为时序分析助手

- 自然语言查询时序数据："上周件量趋势如何？"
- 自动选择合适的预测模型
- 解释预测结果的业务含义

### 方向二：Agent 自主建模

- Agent自动探索特征工程
- Agent自动调参和模型选择
- Agent自动生成预测报告
