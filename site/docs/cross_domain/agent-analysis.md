# Agent 驱动的时序分析

> **核心问题**：如何利用智能体技术提升对时序数据的理解与解释能力？

---

## 🎯 研究方向

### 方向一：自然语言时序查询

用户可以用自然语言提问，Agent 理解意图后：
1. 解析查询意图
2. 从数据库/文件中检索相关时序数据
3. 调用合适的分析方法
4. 用自然语言返回可解释的结果

### 方向二：因果发现与归因

Agent 不仅给出预测值，还能：
- 识别导致异常的关键因素
- 归因到具体事件（促销/天气/节假日）
- 提供反事实分析（"如果没有这次活动会怎样"）
