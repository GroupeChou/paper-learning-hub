---

## 📋 为什么关注交叉领域？

```
时序预测（"预测未来"）
        ×
AI Agent（"智能决策"）
        =
自主决策系统（"预测→决策→行动"的闭环）
```

在顺丰物流场景中：
- **预测Agent** 需要理解时序数据 → 时序能力是Agent的工具之一
- **规划Agent** 的决策需要基于预测结果 → Agent消费时序输出
- **调度系统** 本身就是时序+决策的统一体 → 交叉领域的终极应用
