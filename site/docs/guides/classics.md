# 经典必读

这份清单用于维护“长期基础建设”的论文集合，和每日新增论文分开管理。当前先放一版可直接开始学习的起点，后续可以继续补充、删减和重排。

## 时序预测

1. **Attention Is All You Need**  
   - 机构：Google Brain  
   - 价值：Transformer 基础，后续大部分时序 Transformer 论文都离不开它。
2. **Informer: Beyond Efficient Transformer for Long Sequence Time-Series Forecasting**  
   - 机构：Huawei Cloud / 合作高校  
   - 价值：长序列预测代表作，适合进入时序预测主线。
3. **Autoformer: Decomposition Transformers with Auto-Correlation for Long-Term Series Forecasting**  
   - 机构：合作团队（含产业研究背景）  
   - 价值：分解式建模思路非常适合建立时序论文阅读框架。

## AI Agent

1. **ReAct: Synergizing Reasoning and Acting in Language Models**  
   - 机构：Google Research / Princeton  
   - 价值：Agent 推理 + 行动的经典起点。
2. **Toolformer: Language Models Can Teach Themselves to Use Tools**  
   - 机构：Meta AI  
   - 价值：工具调用能力的经典论文。
3. **Generative Agents: Interactive Simulacra of Human Behavior**  
   - 机构：Google / Stanford  
   - 价值：记忆、规划、反思三段式 Agent 架构的重要来源。

## 使用规则

- 经典论文不追求“最新”，只追求“真的值得反复读”。
- 每读完一篇，建议在条目下追加：核心公式、关键图、工程启发、和当前项目的连接点。
- 如果某篇论文已经被每日增量流程抓到并做了中文精读，可以在这里补一个跳转链接。
