## 💡 核心概念

Agent 记忆体系三层架构：
```
┌─────────────────────────┐
│ 工作记忆 (Working Memory)│ ← 当前对话上下文
├─────────────────────────┤
│ 短期记忆 (Short-term)   │ ← 近期交互历史
├─────────────────────────┤
│ 长期记忆 (Long-term)    │ ← 向量数据库 / 文件系统
│   - 记忆流 Memory Stream │
│   - 反思 Reflection     │
│   - 规划 Plan           │
└─────────────────────────┘
```
