    自动更新机制¶

    本知识库通过 GitHub Actions 实现自动化更新：

    1. 每日定时触发：北京时间 15:00 自动运行
    2. 论文发现：从 arXiv API 抓取白名单机构新论文
    3. WorkBuddy 处理：调用 paper-learning-hub 技能进行中文精读
    4. 自动构建：生成 MkDocs 站点并推送到 GitHub Pages

    详细配置见 run_daily.sh（位于项目根目录）
