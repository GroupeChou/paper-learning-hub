    Markov依赖下多数投票集成的Minimax最优性与谱路由¶

    Minimax Optimality and Spectral Routing for Majority-Vote Ensembles under Markov Dependence

    元信息 内容
    作者 Ibne Farabi Shihab, Sanjeda Akter（爱荷华州立大学 CS Dept）、Anuj Sharma（CE Dept）
    发表时间 2026年4月（预印本，审稿中）
    arXiv ID 2604.13414v1
    领域 集成学习 · 统计学习理论 · 时间序列 · 马尔可夫链
    关键词 Majority-Vote Ensemble, Markov Dependence, Minimax Optimality, Spectral Routing, Fiedler Vector

    摘要（Abstract）¶

    多数投票（Majority-Vote）集成通过对多样化、近似独立的基学习器取平均值来实现方差缩减。但当训练数据呈现马尔可夫依赖（Markov dependence）时——如在时间序列预测、强化学习回放缓冲区和空间网格中——这种经典保证会以现有理论尚未完全量化的方式退化。

    本研究提供了固定维离散分类在马尔可夫设定下的minimax刻画，以及一个在图正则子类上匹配minimax速率的自适应算法。

    三大核心贡献（构成统一的理论弧线）¶

    # 贡要 结论
    1 信息论minimax下界（定理3） 对于混合时间 ≤ \(T_{mix}\) 的平稳可逆几何遍历链，无可测估计器的超额分类风险优于 \(\Omega(\sqrt{T_{mix}/n})\)
    2 均匀Bagging的算法次优性（定理6） AR(1)见证子类上，均匀Bootstrap无法突破 \(\Omega(T_{mix}/\sqrt{n})\)，相对于minimax极限存在 \(\sqrt{T_{mix}}\) 的乘性算法间隙
    3 自适应谱路由闭合间隙（定理10） 通过经验Fiedler特征向量划分训练数据，在图正则子类上达到 \(O(\sqrt{T_{mix}/n}) + \Delta_{geom}\)，无需知道 \(T_{mix}}\)

    实验验证：合成马尔可夫链、2D空间网格、UCR 128数据集档案、Atari DQN集成。

    💡 实践意义：在RL回放缓冲区(\(T_{mix}\approx 50\))上均匀bagging会产生约 \(\sqrt{50}\approx 7\times\) 的超额风险惩罚；谱路由在Atari基准上恢复了大部分差距。

    1 引言（Introduction）¶

    1.1 经典集成理论¶

    集成方法是统计机器学习的基石。其有效性依赖于经典原则：

    对多样化、近似独立的基学习器取平均值可减少方差

    当 \(m\) 个学习器在i.i.d.子样本上训练时： - 集成方差下降：\(O(1/m)\) - 多数投票的超额分类风险下降：\(O(1/\sqrt{m})\) [Breiman, 1996, 2001]

    1.2 现代数据的结构性依赖¶

    然而，现代训练数据是结构性依赖的：

    场景 依赖类型 示例
    时间序列预测 序列依赖 金融/气象预测
    RL集成 时序相关的回放缓冲区 DQN/A2C经验回放
    空间传感器网格 局部相关性 物联网/Wi-Fi指纹

    在马尔可夫依赖下：Bootstrap样本共享重叠信息，而均匀重采样不考虑这一点。这降低了基学习器的有效多样性，以现有i.i.d.分析无法捕捉的方式降低了集成性能。

    1.3 两个核心研究问题¶

    1. 统计极限是什么？ 在马尔可夫依赖下（以混合时间 \(T_{mix}\) 衡量），集成学习的基本统计极限是什么？
    2. 均匀重采样是否达到此极限？ 默认的均匀Bootstrap重采样实践是否接近此极限？还是说盲目重采样与信息论最优之间存在系统性差距？

    1.4 本文定位¶

    现有混合条件下的学习结果[ Yu 1994; Mohri & Rostamizadeh 2008 ]处理回归设置但不刻画多数投票边界的离散分类风险（连续基预测与聚合损失之间介入了非凸阈值）。

    本文将信息论 machinery 适配到非凸多数投票边际的离散分类风险，建立完整的minimax理论。

    2 问题设定（Problem Setting）¶

    2.1 数据模型¶

    设 \(S = \{(X_1, Y_1), ..., (X_n, Y_n)\}\) 从绝对谱间隙 \(\gamma_0 > 0\) 的平稳几何遍历马尔可夫链中抽取，定义在 \(\mathcal{X} \times \{-1, +1\}\) 上。

    • 设 \(T_{mix}\) 表示总变差混合时间
    • \(T_{mix} = \Theta(1/\gamma_0)\)（与谱间隙的关系）

    同质投票集成（大小为 \(m\)）： - 通过随机学习算法在均匀Bootstrap子样本（期望大小 \(n/m\)）上训练基分类器 \(h_1, ..., h_m\) - 聚合预测：\(H(x) = \text{sign}\left(\frac{1}{m}\sum_{i=1}^{m} h_i(x)\right)\)

    2.2 核心挑战：马尔可夫依赖破坏独立性假设¶

    i.i.d. 世界:
  Bootstrap样本 → 近似独立 → 有效多样性高 → 风险 O(1/√m)

马尔可夫依赖世界:
  Bootstrap样本 → 共享重叠信息 → 有效多样性降低 → 风险退化
                              ↑
                          这里的退化程度正是本文要刻画的

    3 主要理论结果（Main Theoretical Results）¶

    3.1 结果一：信息论 Minimax 下界（Theorem 3）¶

    定理3（Minimax Lower Bound）： 对于环境维度 \(d_0\) 固定、混合时间最多为 \(T_{mix}\) 的平稳可逆几何遍历马尔可夫链类 \(\mathcal{P}(d_0)^{T_{mix}}\)： $\(\inf_{\hat{H}} \sup_{P \in \mathcal{P}} R(\hat{H}) \geq \Omega\left(\sqrt{\frac{T_{mix}}{n}}\right)\)$

    即：没有任何可测估计器能达到优于 \(\Omega(\sqrt{T_{mix}/n})\) 的期望超额分类风险。

    证明技术： - 使用应用于高斯AR(1)见证类的广义Fano归约 - 利用AR(1)精度矩阵的三对角结构界定轨迹级KL散度

    3.2 结果二：均匀 Bagging 的算法次优性（Theorem 6）¶

    定理6（Uniform Bagging Suboptimality）： 在我们下界构造底层的AR(1)见证子类上，使用均匀Bootstrap重采样的多数投票集成不能突破超额风险的下界： $\(R(H_{uniform}) \geq \Omega\left(\frac{T_{mix}}{\sqrt{n}}\right)\)$

    相对于minimax极限存在乘性 \(\sqrt{T_{mix}}\) 间隙！

    证明技术： - 通过Le Cam归约在集成的连续边际输出上进行 - 使用Bootstrap基学习器的成对协方差界限（引理5）

    推论7：此障碍扩展到不知晓 \(T_{mix}\) 的固定块方案。

    3.3 结果三：自适应谱路由闭合间隙（Theorem 10）¶

    算法1：自适应谱路由（Adaptive Spectral Routing）

    输入: 训练数据 S = {(Xᵢ,Yᵢ)}ⁿᵢ₌₁, 依赖图 G
输出: 划分后的数据分组 {S₁, S₂, ..., S_P̂}

1. 构建依赖图 G 的邻接/核矩阵
2. 计算经验 Fiedler 特征值 λ̂₂（第二小特征值）
3. 计算 Fiedler 向量 ψ̂₂（对应 λ̂₂ 的特征向量）
4. 设 P̂ = Θ(1/λ̂₂)
5. 沿 Fiedler 向量递归二分，将数据划分为 P̂ 组
6. 每组内部进行Bootstrap采样，训练基学习器
7. 多数投票聚合

    定理10（Spectral Routing Achieves Minimax）： 在 \(n \gg T_{mix}^3 \log n\) 和谱集中引理条件下，所得集成在包含1D路径和有界度空间格子的图正则子类上达到： $\(R(\hat{H}_{spectral}) = O\left(\sqrt{\frac{T_{mix}}{n}}\right) + \Delta_{geom}\)$

    其中 \(\Delta_{geom}\) 是低阶拓扑相关余项。

    🎯 关键优势：算法不需要知道 \(T_{mix}}\)！

    4 直觉解释（Intuition）¶

    为什么均匀Bagging在马尔可夫数据上次优？¶

    时间序列示例 (T_mix ≈ 50):

  数据: ... t₁ t₂ t₃ ... t₄₉ t₅₀ t₅₁ ...
                    ↑             ↑
              Bootstrap样本1    Bootstrap样本2

  问题: t₁和t₄₉虽然相隔近50步，
        但由于马尔可夫依赖仍高度相关!

  均匀Bagging认为它们独立 → 过高估多样性 → 方差缩减不如预期

  谱路由: 用Fiedler向量找到"真正的"数据簇边界
          → 确保同组内的数据真正"足够独立"

    Fiedler向量的直觉¶

    Fiedler向量（拉普拉斯矩阵的第二特征向量）提供了图的最优二分割： - 将图划分为两个子集，使得割边权重最小化 - 在依赖图上 → 将相关性强的数据分开放入不同Bootstrap组 - \(\lambda_2\) 越大 → 图越容易分割 → 需要的组数越少

    7 结论（Conclusion）¶

    本文对马尔可夫依赖下的多数投票集成给出了完整的minimax理论刻画：

    1. 下界: \(\Omega(\sqrt{T_{mix}/n})\) — 不可逾越的信息论极限
    2. 现状: 均匀Bagging只有 \(\Omega(T_{mix}/\sqrt{n})\) — 存在 \(\sqrt{T_{mix}}\) 倍差距
    3. 解法: 自适应谱路由 — 达到 \(O(\sqrt{T_{mix}/n})\)，无需知道 \(T_{mix}\)

    实践建议： - 时间序列/RL/空间数据上的集成学习 → 不要用均匀Bootstrap - 使用谱路由（或其他依赖感知的重采样方案） - 特别是在混合时间长（\(T_{mix}\)大）的场景下收益显著
